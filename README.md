# OpTrack – Smart Opportunity Intelligence for Students

## Overview
OpTrack is a web-based application designed to help students discover and **prioritize opportunities** such as internships, hackathons, and scholarships.

Unlike traditional platforms, OpTrack focuses on **decision-making**, enabling users to identify the most relevant opportunities based on their skills, deadlines, and preferences.

## Problem Statement
Students often face difficulty in:
- Finding relevant opportunities
- Managing multiple deadlines
- Deciding which opportunities to apply for

This leads to missed deadlines and inefficient applications.

## Solution
OpTrack solves this problem by:
- Providing a centralized dashboard for opportunities
- Introducing a **Match Score system** to measure relevance
- Highlighting **deadline urgency**
- Allowing users to **filter, sort, and search** efficiently

## Key Features (Planned)

###  Search
Users can search opportunities by title, company, or skills.

### Filtering
- Filter by domain (Tech, Design, etc.)
- Filter by stipend range
- Filter by deadline

### Sorting
- Sort by deadline
- Sort by stipend
- Alphabetical sorting

### Match Score (Unique Feature)
Shows how well an opportunity matches the user's skills.

### ⏳ Deadline Indicator
Displays urgency using visual indicators:
- Urgent(RED)
- Moderate(YELLOW)
- Safe(GREEN)

### Save Opportunities
Users can bookmark opportunities using local storage.

### Dark Mode
Toggle between light and dark themes.
